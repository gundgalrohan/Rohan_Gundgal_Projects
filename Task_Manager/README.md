Task Manager
A Java console application for managing personal tasks with priority levels, filtering, and sorting.

Features
Add tasks with title and priority (Low / Medium / High)
View, edit, and delete tasks
Mark tasks as completed
Filter tasks by completion status
Sort tasks by priority level

Tech Stack
Java ArrayList Stream API Scanner

How to Run
bashjavac TaskManager.java
java TaskManager

Menu Options
Add Task
View All Tasks
Mark Task as Completed
Delete Task
Edit Task
Filter Tasks by Status
Sort Tasks by Priority
Exit
