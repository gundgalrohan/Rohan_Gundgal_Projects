Hypopigmentation Detector
A browser-based symptom checker that identifies likely hypopigmentation disorders using symptom matching with percentage-based confidence scoring.

Detectable Conditions
Vitiligo
Albinism
Pityriasis Alba
Tinea Versicolor
Hypomelanosis

Features
Searchable, filterable symptom checklist
Percentage match scoring across all disorders
Precautions and care tips for the top matched condition
Fully client-side — no backend required

Tech Stack
HTML CSS Vanilla JavaScript JSON

How to Run
Open index.html in any browser.
Ensure symptoms_data.json and precaution_data.json are in the same directory.
