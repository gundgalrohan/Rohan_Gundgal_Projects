Snake Game
A feature-rich Snake game built with Pygame featuring stone obstacles, a live leaderboard, smooth snake rendering, and pause functionality.

Features
Classic snake gameplay with apple collection
Stone obstacle strips randomly placed each round
Top-5 high score leaderboard (session-based)
Pause/resume with ESC
Animated snake head with eyes, pupils, and tongue
Apple with stem graphic

Tech Stack
Python Pygame

How to Run
bashpip install pygame
python snake_game.py

Controls
Key        Action
W / ↑    Move Up
S / ↓    Move Down
A / ←    Move Left
D / →    Move Right
ESC      Pause / Resume
R        Restart
